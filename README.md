# LDR_light


Connections

LDR sesnor to Arduino
Sensor D0 pin = Arduino digital pin 7
Sensor GND pin = Arduino GND pin
Sensor VCC pin = Arduino 5v pin
LED to Arduino
LED VCC pin = Arduino digital pin 13
LED GND pin = Arduino GND pin
